* Abstract

This document presents a systematic verification of the variance decomposition method for assessing parameter influence in simulation systems. Through three carefully designed experiments, we demonstrate the method's ability to correctly distinguish between cases where measurement outputs are controlled by input parameters versus cases where they are not. The controlled experiments provide empirical validation of the theoretical framework and establish confidence bounds for practical applications.

* Introduction

Parameter influence assessment in complex simulation systems requires robust statistical methods that can separate genuine parameter effects from measurement noise. This document provides rigorous verification of the variance decomposition approach through controlled synthetic experiments where ground truth is known a priori.

**Verification Objectives:**
1. Validate method performance under known parameter control conditions
2. Confirm method sensitivity to absence of parameter control  
3. Demonstrate applicability to trajectory prediction scenarios

The verification employs synthetic data generation with controlled noise and parameter relationships, enabling quantitative assessment of method accuracy and reliability.

** Theoretical Framework

The variance decomposition method leverages the law of total variance to quantify parameter influence:
\[
\operatorname{Var}(H) = E[\operatorname{Var}(H|X)] + \operatorname{Var}(E[H|X])
\]

**Measurement Protocol:**
- **Same parameter variance**: $\operatorname{Var}(H - H') | X = x \approx 2E[\operatorname{Var}(H|X)]$ (measurement noise)
- **Random parameter variance**: $\operatorname{Var}(H(X) - H'(X')) \approx 2E[\operatorname{Var}(H|X)] + 2\operatorname{Var}(E[H|X])$ (noise + parameter effect)

**Impact Variance**: The difference between these quantities directly quantifies parameter control magnitude.
